# 网易静态博客
实习初期学习过程中的作业的作品，使用原生的JavaScript。
###亮点
    * 工具类函数的模块化封装，基于**保留模块模式**
    * 规范的注释、类名、符合**NEC**的规范
    * 滚动静止的侧边栏函数封装
    * 函数的松散耦合，降低函数之间的依赖性
###兼容性
未刻意考虑浏览器的兼容性，目前支持IE10+以及主流浏览器
